Customize the Ball

You can make the ball move whichever directions you want by editing the BALL PATH section in the HTML code. Open this file in any text editor and look for the section marked below:

// ========================================
// BALL PATH - EDIT THIS SECTION
// Change the waypoints to control where
// the ball goes. Each point is [x, y].
// Set useCustomPath to true to enable.
// ========================================

const useCustomPath = false;

const ballPath = [
  [100, 100],   // top-left
  [500, 200],   // middle-right
  [300, 400],   // center-bottom
  [700, 100],   // top-right
  [200, 500],   // bottom-left
];

// Speed of movement between points (1-10)
const pathSpeed = 3;

When useCustomPath is set to true, the ball will visit each waypoint in order. When false, it bounces randomly as usual.

On mobile devices, you can also tilt your device to steer the ball directly.
